## 📁 Список лабораторных работ

### Лабораторная работа №1
**Работа с непрямоугольными матрицами**
Реализация алгоритмов эффективного хранения и обработки разреженных и непрямоугольных матриц.

---

### Лабораторная работа №2
**Элементарные структуры данных**
Работа со стеком, очередью и двусторонней очередью (деком). Реализация основных операций и управление динамической памятью.

---

### Лабораторная работа №3: Таблицы
**Часть А: Просматриваемые и упорядоченные таблицы**
Реализация структур словарей с использованием алгоритмов линейного и бинарного поиска.  
**Часть Б: Хеш-таблицы**
Разработка хеш-таблиц с различными стратегиями разрешения коллизий (открытая адресация или метод цепочек).

---

### Лабораторная работа №4: Деревья
**Часть А: Двоичное дерево поиска (BST)**
Реализация базовых операций: вставка, удаление и различные методы обхода дерева.  
**Часть Б: Самобалансирующиеся деревья поиска**
Изучение и реализация сбалансированных структур (например, AVL или красно-черные деревья) для обеспечения производительности $O(\log n)$.

---

### Лабораторная работа №5
**Работа с графами**
Представление графов (списки/матрицы смежности) и реализация фундаментальных алгоритмов (BFS, DFS, алгоритм Дейкстры и др.).

---

## 📊 Анализ производительности
Для большинства работ проведено таймирование и анализ сложности:
* **Таймирование:** Замеры времени выполнения для различных объемов входных данных ($n$).
* **Данные:** Результаты бенчмарков сохранены в формате `.csv`.
* **Визуализация:** Скрипты на Python (`.py`) использовались для построения графиков и сравнения теоретической сложности с эмпирическими данными.
